# AI_WORKFLOW

## Regra de trabalho

- Codex Terminal: implementa no codigo real.
- ChatGPT Web/Mobile: planeja, revisa texto e estrategia.
- Copilot: sugestao local no editor.

## Fluxo curto

1. Definir objetivo em 1 linha.
2. Executar no codigo (Codex).
3. Validar com comandos.
4. Registrar handoff no fim.

## Regras operacionais

- Nao trabalhar direto em `main` para mudancas grandes.
- Sempre validar antes de push.
- Sempre atualizar docs quando comportamento mudar.
